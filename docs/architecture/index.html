Architecture

The following diagram demonstrates the overall cluster design which will be implemented via the Ansible automation if you follow the above guide.

To obtain a highly available Kubernetes cluster we've chosen to use the stacked etcd toplogy. This is the default configuration as the external etcd cluster alternative requires additional compute resources.
